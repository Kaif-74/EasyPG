# EasyPG - Find Your Perfect PG in Minutes

**EasyPG** is a professional PG (Paying Guest) Search and Vacancy Management System. It allows students and working professionals to find verified accommodations easily while providing PG owners with a robust dashboard to manage their properties, rooms, and real-time vacancies.

## 🌐 Live Demo
Check out the live project here: [http://shaikaif.pythonanywhere.com](http://shaikaif.pythonanywhere.com)

## ✨ Key Features
- **Advanced Search:** Filter PGs by City, Area, Category (Men/Women/Co-Living), Rent Range, and Sharing Type.
- **Owner Dashboard:** Private management area for property owners to add/edit/delete PGs.
- **Room Management:** Real-time vacancy updates and pricing management for individual rooms.
- **Dynamic Gallery:** Multiple image upload support for both buildings and specific rooms.
- **Super Admin Panel:** Custom dashboard for site administrators to monitor platform growth and manage all listings.
- **Responsive Design:** Fully mobile-friendly UI built with Pure CSS.

## 🛠️ Technologies Used
- **Backend:** Python, Django 6.0
- **Database:** MySQL (Local) / SQLite (Production)
- **Frontend:** HTML5, CSS3 (Custom Grid & Flexbox)
- **Deployment:** PythonAnywhere
